# Excel Chatbot AI 🤖📊

An intelligent AI-powered chatbot application for Excel operations, calculations, and file generation with image-to-Excel conversion capability.

## 🎯 Features

✨ **Core Features**
- 🤖 AI Chatbot for Excel guidance and operations
- 📊 Automatic Excel file generation (.xlsx)
- 🖼️ Image to Excel conversion (OCR)
- 📝 Formula assistance and calculations
- 💰 Pay-per-click monetization tracking
- 💧 Automatic watermarking ("nurasyid")
- 💻 Desktop app installation capability
- 📥 One-click Excel download

## 🚀 Quick Start

### Installation
```bash
git clone https://github.com/Lovabit-RGB/excel-chatbot-ai.git
cd excel-chatbot-ai
npm install
```

### Run Development Server
```bash
npm start
```

### Build Desktop App
```bash
npm run electron-build
```
